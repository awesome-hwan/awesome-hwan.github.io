# Angular 2 개발환경 구축하기  

> 플러그인 설치  

`command + ,`를 눌러 Settings 창을 열어서 Atom TypeScript를 설치한다.  

> Cil 설치  

통합 개발환경이다. 

`npm install angular-cli -g`

> Angular를 이용하여 디렉토리 생성  

`ng new first-app`  
해당 파일에 node_modules이 전부 설치되면 자동으로 완료된다.  

> Angular 2 실행하기  

모두 설치가 완료 되었다면
`cd first-app`을 이용해 디렉토리로 이동 후 

`ng serve`를 입력하자
